# Care vs Endurance — A Digital Humanities Comparative Study of Han Kang & László Krasznahorkai

**Theme:** A comparative exploration of two acclaimed authors’ literary visions, analyzing convergences and contrasts in their portrayals of **human fragility**, **moral endurance**, and the **state of the modern world** — revealing what their works disclose about humanity’s current direction.

_By Eprene_

## What this project does
- Builds a small comparative corpus (Han Kang vs László Krasznahorkai)
- Runs mixed‑method analysis: frequency, sentiment (lexicon‑based), lexical diversity, theme hits, keyword extraction, and a co‑occurrence network
- Generates charts in `output/` and a Pages‑ready site in `docs/` with interpretation

## How to run (locally)
```bash
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python code/analyze_care_endurance_project.py
```
Outputs will appear in `output/` and are embedded in `docs/index.md` (after you enable GitHub Pages).
